# Canvascope Newsroom

Static newsroom for Canvascope, a privacy-first AI study workspace for students.

Canonical site: https://canvascope-blog.vercel.app/

Canvascope combines LMS search, grounded course AI, Smart Planner, Student Profiles, Lectra iPad PDF annotation, and DropBridge file handoff so students can find course material, study it, annotate it, and bring finished work back into browser workflows.
